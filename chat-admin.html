Cara Menghubungi Admin DIRASTORE

Admin kami siap membantu Anda dengan pertanyaan atau pembelian. Ikuti langkah-langkah di bawah ini untuk mulai chat dengan admin.

  1. 1. Klik Tombol Chat Admin

    Pada halaman utama atau halaman ini, Anda dapat menemukan tombol "Chat Admin".

  2. 2. Masukkan Pesan Anda

    Setelah diarahkan ke WhatsApp, masukkan pesan Anda. Misalnya: "Halo, saya ingin bertanya tentang produk."

  3. 3. Tunggu Respon

    Admin kami akan segera merespon. Harap bersabar jika antrian sedang penuh.
